*8 metros: Uma medição padrão de 8 metros foi feita usando uma régua a laser, após a qual o software de estabilidade foi usado para ler a distância de teste de o radar triangular Delta-2. De acordo com os resultados do teste, o alcance máximo é de 8 metros. O relatório do teste foi produzido pelo Laboratório Shanchuan.
*6x360°/s: Um tacômetro foi usado para testar a velocidade e frequência de rotação do radar. A velocidade medida foi ≥360rpm e a frequência de varredura foi de 6Hz±0,1Hz. O relatório do teste foi produzido pelo Laboratório Shanchuan.
*2K/s: O software de estabilidade foi usado para testar a taxa de amostragem em uma área de teste de 1,7*1,7 metros. O resultado foi maior que 2K/s. O relatório do teste foi produzido pelo Laboratório Shanchuan.
*2,5%: Uma régua a laser foi usada para medir as distâncias padrão de 5,5 metros e 8 metros, depois um software de estabilidade foi usado para ler a distância de teste do radar triangular Delta-2 e calcular a precisão. De acordo com os resultados do teste, a precisão foi de -2,50% a 2,50%. O relatório do teste foi produzido pelo Laboratório Shanchuan.
*Sucção de 4000Pa: Foi aplicada uma voltagem de 16,5V e uma máquina de sucção foi usada para testar a sucção a 8%, 12%, 19%, 49% e 100% de potência. Os resultados dos testes mostraram que o maior valor de sucção foi de 4000Pa. O relatório do teste foi produzido pelo Laboratório Shanchuan. *3200mAh é a capacidade nominal da bateria.
*130 minutos: O robô foi totalmente carregado, definido para o modo de limpeza e funcionou em um ambiente aberto até que voltasse a carregar. Os resultados do teste mostraram que o robô tem uma duração de bateria de 130 minutos ±10% no modo Padrão. O relatório do teste foi produzido pelo Laboratório Shanchuan.
